== Description ==

IM Yell was designed to allow Bloggers and Internet Marketers a quick way to select text in the visual editor and give it a colored background.  It now uses the native mceTiny background color selector and icon.



== Installation ==

1. Upload the entire folder 'im-yell' to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
